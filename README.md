# Climate Emissions Analysis

This project analyses global CO2 emissions data using Python to identify key trends and major contributing countries.
# Objectives
- Explore global emissions data
- Identify top emitting countries
- Understand distribution of emissions
# Methods
- Data cleaning and preprocessing using Pandas
- Handling missing values and type conversion
- Sorting and filtering datasets
- Data visualisation using Matplotlib and Seaborn
# Key Insights
- Emissions are highly concentrated among a small number of countries
- China and the United States are the largest contributors
- Emerging economies like India show increasing emissions
#Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
# Output
Bar chart showing top 10 emitting countries in 2023
